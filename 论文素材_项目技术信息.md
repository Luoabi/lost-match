# 校园失物追寻平台 - 技术素材文档

> 本文档仅包含客观技术事实，用于论文撰写参考

## 一、项目基本信息

### 1.1 项目名称
- 后端项目名称：BRApi (campus-lost-found-api)
- 前端项目名称：admin (校园失物追寻平台 - 后台管理系统)
- 项目类型：校园失物招领管理系统

### 1.2 系统角色
- student（学生）：普通用户角色
- teacher（教师）：普通用户角色
- admin（管理员）：系统管理员角色
- cleaner（保洁员）：特殊用户角色

### 1.3 后端技术栈
| 技术/框架 | 版本 | 用途 |
|---------|------|------|
| Spring Boot | 3.2.0 | 后端应用框架 |
| Java | 17 | 开发语言 |
| Spring Data JPA | 3.2.0 | ORM框架 |
| MySQL Connector | runtime | 数据库驱动 |
| Lombok | optional | 代码简化工具 |
| Spring Boot Validation | 3.2.0 | 数据验证 |
| JJWT | 0.11.5 | JWT令牌生成与验证 |
| Hutool | 5.8.23 | Java工具类库 |
| SpringDoc OpenAPI | 2.2.0 | API文档生成 |
| Hibernate | 6.x | JPA实现 |

### 1.4 前端技术栈
| 技术/框架 | 版本 | 用途 |
|---------|------|------|
| Vue | 3.5.32 | 前端框架 |
| Element Plus | 2.13.6 | UI组件库 |
| Vue Router | 4.6.4 | 路由管理 |
| Pinia | 3.0.4 | 状态管理 |
| Pinia Plugin Persistedstate | 4.7.1 | 状态持久化 |
| Axios | 1.14.0 | HTTP请求库 |
| Mock.js | 1.1.0 | 模拟数据 |
| NProgress | 0.2.0 | 进度条组件 |
| Vite | 8.0.4 | 构建工具 |
| @element-plus/icons-vue | 2.3.2 | 图标库 |

### 1.5 数据库信息
- 数据库类型：MySQL
- 字符集：utf8mb4
- 排序规则：utf8mb4_unicode_ci
- 数据库名称：lost_found（或 campus_lost_found）
- 时区设置：Asia/Shanghai

### 1.6 服务器配置
- 后端端口：8080
- 上下文路径：/api
- JWT密钥：campus-lost-found-secret-key-2025
- JWT过期时间：86400000毫秒（24小时）
- 文件上传路径：/uploads/
- 最大文件大小：5242880字节（5MB）
