# 个人中心功能说明

## 功能概述
在后台管理系统的头部导航栏右侧，点击用户头像下拉菜单中的"个人中心"，可以打开个人中心弹窗。

## 功能特性

### 1. 个人信息管理
- **头像上传**
  - 支持jpg、png等图片格式
  - 建议尺寸：100x100px
  - 文件大小限制：2MB以内
  - 点击头像区域即可选择上传
  - 上传成功后实时预览

- **基本信息编辑**
  - 用户名：可修改，但不能与其他用户重复
  - 邮箱：可修改
  - 手机号：可修改，但不能与其他用户重复
  - 角色：只读显示（管理员/普通用户）
  - 注册时间：只读显示

### 2. 修改密码
- **密码修改流程**
  - 输入原密码进行验证
  - 输入新密码（至少6位）
  - 再次确认新密码
  - 修改成功后自动退出登录，需重新登录

- **安全提示**
  - 密码修改后需要重新登录
  - 原密码验证失败会提示错误
  - 两次输入的新密码必须一致

### API接口

#### 1. 更新个人信息
```
PUT /user/profile
Content-Type: application/json

{
  "id": 1,
  "username": "newusername",
  "email": "newemail@example.com",
  "phone": "13800138000",
  "avatar": "http://example.com/avatar.jpg"
}
```

#### 2. 修改密码
```
PUT /user/change-password
Content-Type: application/json

{
  "id": 1,
  "oldPassword": "oldpass123",
  "newPassword": "newpass123"
}
```

## 使用流程

### 修改个人信息
1. 点击右上角用户头像
2. 选择"个人中心"
3. 在"个人信息"标签页中：
   - 点击头像区域上传新头像
   - 修改用户名、邮箱、手机号
   - 点击"保存信息"按钮
4. 保存成功后，页面会自动更新用户信息

### 修改密码
1. 点击右上角用户头像
2. 选择"个人中心"
3. 切换到"修改密码"标签页
4. 依次输入：
   - 原密码
   - 新密码（至少6位）
   - 确认新密码
5. 点击"修改密码"按钮
6. 修改成功后会自动退出登录
7. 使用新密码重新登录

## 安全特性

1. **密码加密**：使用MD5加密存储
2. **原密码验证**：修改密码时必须验证原密码
3. **唯一性检查**：用户名和手机号不能重复
4. **权限控制**：用户只能修改自己的信息
5. **自动登出**：密码修改后强制重新登录

## 注意事项

1. 头像上传需要后端文件上传接口支持
2. 密码修改后会清除本地登录状态
3. 用户名和手机号修改时会检查是否被占用
4. 所有操作都有相应的成功/失败提示
